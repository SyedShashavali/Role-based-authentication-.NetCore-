Project Overview: Role-Based Authentication in ASP.NET Core
Repository: SyedShashavali/Role-based-authentication-.NetCore-

Key Details:
Primary Language: C#
Framework: ASP.NET Core
Owner: Syed Shashavali
Repository Status: Active
Visibility: Public
Project Description:
This is an ASP.NET Core application that demonstrates:

Role-Based Authentication: Implementation of user authentication with different role levels
Role-Based Authorization: Authorization logic that restricts access based on user roles
CRUD Operations: Complete Create, Read, Update, and Delete functionality for managing employee records
